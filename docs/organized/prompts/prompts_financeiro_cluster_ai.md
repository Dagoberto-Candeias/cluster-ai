# 💰 Catálogo de Prompts para Análises Financeiras - Cluster-AI

## 🎯 Guia Rápido de Utilização

### Configurações Recomendadas
- **Temperatura Baixa (0.1-0.3)**: Para cálculos precisos, projeções conservadoras
- **Temperatura Média (0.4-0.6)**: Para análises de cenários, recomendações
- **Temperatura Alta (0.7-0.9)**: Para cenários hipotéticos, inovação financeira

### Modelos por Categoria
- **LLM Geral**: Mixtral, Llama 3
- **Análise Quantitativa**: Mixtral, Llama 3 70B

## 📁 CATEGORIA: ANÁLISE FINANCEIRA GERAL

### 1. Análise de Custos Operacionais
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um analista financeiro sênior]

Realize análise completa de custos operacionais do Cluster-AI:

**Estrutura de custos atual:**
- Infraestrutura: [servidores, cloud, energia]
- Pessoal: [equipe técnica, administrativa]
- Software/Licenças: [ferramentas, plataformas]
- Marketing/Vendas: [campanhas, eventos]
- Outros: [escritório, viagens, etc.]

**Dados de uso:** [métricas de utilização, crescimento]
**Período de análise:** [mês/trimestre/atual]

**Solicito:**
1. **Breakdown detalhado** de custos por categoria
2. **Análise de eficiência** e identificação de ineficiências
3. **Benchmarking** com padrões do mercado
4. **Projeções de otimização** de custos
5. **ROI (Retorno sobre Investimento)** por iniciativa
```

### 2. Projeção de Receitas e Cenários
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um analista de receita e pricing]

Desenvolva projeções de receita para o Cluster-AI:

**Modelo de negócio atual:** [open source/freemium/enterprise/SaaS]
**Base de usuários:** [ativos, crescimento mensal, churn]
**Produtos/Serviços:** [linha completa, preços atuais]
**Mercado alvo:** [segmentação, TAM/SAM/SOM]

**Solicito:**

3. **Projeções de crescimento** por segmento
4. **Análise de churn** e estratégias de retenção

## 📁 CATEGORIA: GESTÃO DE CAPITAL DE GIRO

### 4. Análise de Capital de Giro
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um analista de working capital]

Analise e otimize o capital de giro do Cluster-AI:

**Componentes atuais:**
- Contas a receber: [valor médio, prazo médio]
- Estoques: [se aplicável - modelos, licenças]
- Contas a pagar: [prazo médio, condições]
- Caixa disponível: [liquidez imediata]

**Ciclos operacionais:** [recebimento, pagamento, produção]

## 💡 DICAS PARA ANÁLISE FINANCEIRA

### Precisão nos Dados
Sempre valide a qualidade e consistência dos dados financeiros

### Cenários Múltiplos
Considere sempre múltiplos cenários (conservador/base/otimista)

### Foco no Cash Flow
O fluxo de caixa é mais importante que o lucro contábil

### Sensibilidade
Analise como mudanças em variáveis afetam os resultados

### Comunicação Clara
Apresente análises complexas de forma clara e acionável
