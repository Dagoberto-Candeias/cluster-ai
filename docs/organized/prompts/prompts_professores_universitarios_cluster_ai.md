# 🎓 Catálogo de Prompts para Professores Universitários

## 🎯 Guia Rápido de Utilização

### Configurações Recomendadas
- **Temperatura Baixa (0.1-0.3)**: Para planejamento pedagógico estruturado, avaliações
- **Temperatura Média (0.4-0.6)**: Para criação de conteúdo, exercícios práticos
- **Temperatura Alta (0.7-0.9)**: Para geração de ideias criativas, projetos inovadores

### Modelos por Categoria
- **LLM Geral**: Llama 3, Mixtral (planejamento, conteúdo)
- **Código e Tutoriais**: CodeLlama, Qwen2.5-Coder (exercícios práticos)
- **Especializados**: DeepSeek-Coder (projetos avançados)

---

## 📁 CATEGORIA: PLANEJAMENTO PEDAGÓGICO

### 1. Planejamento de Disciplina
**Modelo**: Llama 3/Mixtral

```
[Instrução: Atue como um especialista em planejamento pedagógico universitário]

Crie um plano completo de disciplina para o nível universitário:

**Informações da disciplina:**
- **Nome da disciplina:** [ex: Algoritmos e Estruturas de Dados]
- **Curso:** [ex: Ciência da Computação]
- **Carga horária:** [ex: 60 horas]
- **Pré-requisitos:** [ex: Programação Básica]
- **Público-alvo:** [graduação/iniciação científica/pós-graduação]

**Solicito:**
1. **Objetivos de aprendizagem** claros e mensuráveis
2. **Ementa detalhada** com tópicos e subtópicos
3. **Metodologia de ensino** adequada ao conteúdo
4. **Avaliação contínua** e formativa
5. **Recursos necessários** (livros, softwares, laboratórios)
6. **Cronograma semanal** com atividades práticas
```

### 2. Plano de Aula Individual
**Modelo**: Llama 3/Mixtral

```
[Instrução: Atue como um professor experiente em planejamento de aulas]

Desenvolva um plano de aula detalhado e executável:

**Contexto da aula:**
- **Disciplina:** [ex: Inteligência Artificial]
- **Tema:** [ex: Redes Neurais Artificiais]
- **Duração:** [ex: 2 horas]
- **Nível:** [graduação/pós-graduação]
- **Pré-requisitos:** [conhecimentos necessários]

**Solicito:**
1. **Objetivos específicos** da aula
2. **Conteúdo programático** estruturado
3. **Metodologias ativas** (problematização, estudo de caso)
4. **Recursos audiovisuais** e materiais de apoio
5. **Atividades práticas** com tempo estimado
6. **Avaliação da aprendizagem** durante a aula
7. **Referências bibliográficas** atualizadas
```

### 3. Projeto de Ensino Inovador
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um pedagogo inovador especializado em educação superior]

Crie um projeto de ensino que incorpore metodologias ativas e tecnologias:

**Parâmetros do projeto:**
- **Disciplina/área:** [ex: Data Science, Engenharia, Medicina]
- **Duração:** [semestral/anual/projeto específico]
- **Número de alunos:** [individual/grupo/classe]
- **Recursos disponíveis:** [laboratório, software, equipamentos]

**Solicito:**
1. **Justificativa pedagógica** do projeto
2. **Objetivos educacionais** inovadores
3. **Metodologias ativas** (PBL, flipped classroom, gamificação)
4. **Integração de tecnologias** (IA, realidade virtual, plataformas online)
5. **Avaliação formativa** e somativa
6. **Indicadores de sucesso** mensuráveis
7. **Plano de sustentabilidade** do projeto
```

## 📁 CATEGORIA: DESENVOLVIMENTO DE CONTEÚDO

### 4. Material Didático Interativo
**Modelo**: CodeLlama/Qwen2.5-Coder

```
[Instrução: Atue como um designer instrucional especializado em materiais digitais]

Desenvolva material didático interativo para ensino universitário:

**Especificações:**
- **Tema:** [ex: Algoritmos de Ordenação]
- **Formato:** [apresentação/webpage/interativo]
- **Público:** [graduação/pós-graduação]
- **Duração estimada:** [30-60 minutos]

**Solicito:**
1. **Estrutura hierárquica** do conteúdo
2. **Elementos interativos** (quizzes, simulações)
3. **Código fonte** para implementações práticas
4. **Exemplos reais** de aplicação
5. **Exercícios autoavaliativos** com feedback
6. **Referências atualizadas** e leituras complementares
```

### 5. Exercícios e Atividades Práticas
**Modelo**: CodeLlama/DeepSeek-Coder

```
[Instrução: Atue como um especialista em avaliação formativa]

Crie uma bateria de exercícios práticos para reforço de aprendizagem:

**Contexto:**
- **Disciplina:** [ex: Machine Learning]
- **Tópico específico:** [ex: Regressão Linear]
- **Nível de dificuldade:** [básico/intermediário/avançado]
- **Tipo de exercício:** [programação/problema teórico/análise de caso]

**Solicito:**
1. **Enunciado claro** e objetivo
2. **Contexto prático** de aplicação
3. **Dicas progressivas** para resolução
4. **Solução comentada** passo-a-passo
5. **Extensões** para alunos avançados
6. **Rubrica de avaliação** detalhada
```

### 6. Estudos de Caso Acadêmicos
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um especialista em metodologias ativas de ensino]

Desenvolva um estudo de caso completo para discussão em sala de aula:

**Tema do caso:**
- **Área:** [ex: Tecnologia, Negócios, Saúde]
- **Complexidade:** [técnica/estratégica/ética]
- **Duração da discussão:** [1-2 aulas]

**Solicito:**
1. **Cenário realista** baseado em casos reais
2. **Problemas múltiplos** para análise
3. **Dados quantitativos** e qualitativos
4. **Questões orientadoras** para discussão
5. **Soluções alternativas** possíveis
6. **Material de apoio** (gráficos, tabelas)
7. **Guia para o professor** com pontos-chave
```

## 📁 CATEGORIA: AVALIAÇÃO E FEEDBACK

### 7. Sistema de Avaliação Contínua
**Modelo**: Llama 3/Mixtral

```
[Instrução: Atue como um especialista em avaliação educacional]

Desenvolva um sistema completo de avaliação para uma disciplina:

**Parâmetros:**
- **Disciplina:** [nome e área]
- **Carga horária:** [total e distribuição]
- **Perfil dos alunos:** [graduação/pós-graduação]

**Solicito:**
1. **Matriz de competências** a serem avaliadas
2. **Diversidade de instrumentos** (provas, projetos, participação)
3. **Peso relativo** de cada componente
4. **Critérios de avaliação** claros e objetivos
5. **Feedback formativo** durante o semestre
6. **Mecanismos de recuperação** de aprendizagem
7. **Análise de resultados** e melhoria contínua
```

### 8. Rubricas de Avaliação Detalhadas
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um especialista em avaliação por competências]

Crie rubricas analíticas para avaliação de diferentes tipos de trabalho:

**Tipo de avaliação:**
- **Trabalho:** [pesquisa/projeto/programação/apresentação]
- **Critérios principais:** [conteúdo, metodologia, apresentação, inovação]
- **Níveis de desempenho:** [insuficiente, regular, bom, excelente]

**Solicito:**
1. **Descrição detalhada** de cada critério
2. **Indicadores específicos** de desempenho
3. **Exemplos concretos** para cada nível
4. **Peso relativo** dos critérios
5. **Orientações para alunos** sobre expectativas
6. **Guia para correção** consistente
```

### 9. Análise de Desempenho dos Alunos
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um analista educacional]

Analise o desempenho dos alunos e proponha intervenções pedagógicas:

**Dados disponíveis:**
- **Notas por atividade:** [provas, trabalhos, participação]
- **Perfil da turma:** [background, dificuldades comuns]
- **Objetivos da disciplina:** [competências esperadas]

**Solicito:**
1. **Diagnóstico detalhado** do desempenho coletivo
2. **Identificação de gaps** de aprendizagem
3. **Intervenções específicas** por grupo de alunos
4. **Estratégias de recuperação** individualizadas
5. **Ajustes no planejamento** pedagógico
6. **Recomendações para próximos semestres**
```

```

3. **Estrutura do trabalho** (capítulos, seções)

5. **Técnicas de coleta** e análise de dados
6. **Critérios de qualidade** acadêmica
7. **Orientações éticas** da pesquisa
```

**Parâmetros:**

## 💡 RECURSOS PEDAGÓGICOS RECOMENDADOS

### Plataformas Educacionais
- **Moodle/Open edX** - LMS para gestão de cursos
- **Google Classroom/Microsoft Teams** - Colaboração e comunicação
- **Jupyter Notebooks** - Ensino de programação e análise de dados
- **GitHub Classroom** - Gestão de projetos e versionamento

### Ferramentas de Avaliação
- **Peergrade** - Avaliação entre pares
- **Kahoot/Quizizz** - Avaliação interativa
- **Rubistar** - Criação de rubricas
- **Gradescope** - Correção automatizada

### Recursos de Inovação
- **Flipped Classroom** - Metodologia invertida
- **Gamificação** - Elementos de jogos na aprendizagem
- **Realidade Virtual** - Experiências imersivas
- **IA Educacional** - Personalização da aprendizagem
