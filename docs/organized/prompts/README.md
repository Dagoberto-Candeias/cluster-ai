## 🚀 Como Usar

1. **Identifique seu perfil**: Escolha o arquivo de prompts correspondente à sua função
2. **Selecione o cenário**: Cada arquivo contém prompts para diferentes cenários
3. **Configure no Ollama**: Use os prompts como system prompts
4. **Integre com IDEs**: Importe as personas para VSCode, PyCharm, etc.

## 💡 Dicas de Uso

- **Temperatura**: Ajuste conforme a tarefa (0.2-0.4 para código, 0.4-0.6 para análise)
- **Modelo**: Use CodeLlama para desenvolvimento, Mixtral para análise geral
- **Contexto**: Sempre forneça contexto específico do Cluster-AI

## 🔗 Integrações Disponíveis

- **Ollama**: System prompts nativos
- **OpenWebUI**: Personas especializadas
- **VSCode/Cursor**: Templates de prompt
- **Outras IDEs**: Arquivos JSON de configuração
