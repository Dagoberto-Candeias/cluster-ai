# 📚 Documentação Organizada - Cluster-AI

## 📁 Estrutura da Documentação

Esta pasta contém toda a documentação do projeto Cluster-AI organizada de forma estruturada e fácil de navegar.

### 🗂️ Organização por Categoria

#### 📦 `installation/` - Instalação e Setup
- `ANDROID_GUIA_RAPIDO.md` - Guia rápido de instalação Android
- `ANDROID_INSTALACAO_WORKER.md` - Instalação detalhada de workers Android
- `ANDROID.md` - Documentação completa Android
- `INSTALACAO.md` - Guia geral de instalação

#### 🚀 `usage/` - Uso e Operação
- `OLLAMA.md` - Como usar o Ollama no Cluster-AI
- `OPENWEBUI.md` - Interface web OpenWebUI

#### 🔧 `maintenance/` - Manutenção e Configuração
- `BACKUP.md` - Sistema de backup e restauração
- `CONFIGURACAO.md` - Configurações avançadas

#### 📋 `prompts/` - Prompts e Integrações
- Todos os arquivos de prompts especializados por categoria
- `openwebui_integration.md` - Integração com OpenWebUI
- `openwebui_personas.json` - Personas para OpenWebUI

#### 🛠️ `development/` - Desenvolvimento
- `arquitetura.txt` - Arquitetura do sistema
- Guias de desenvolvimento e troubleshooting

#### 🔒 `security/` - Segurança
- `SECURITY_MEASURES.md` - Medidas de segurança implementadas

#### 📋 `guides/` - Guias Gerais
- Guias de uso, otimização, troubleshooting
- Documentação de funcionalidades específicas

#### 📝 `templates/` - Templates
- `script_comments_template.sh` - Template para comentários de scripts

## 🎯 Como Navegar

1. **Para instalação**: Comece com `installation/ANDROID_GUIA_RAPIDO.md`
2. **Para uso diário**: Consulte `usage/` e `guides/`
3. **Para desenvolvimento**: Veja `development/` e `prompts/`
4. **Para manutenção**: Use `maintenance/` e `security/`

## 📖 Documentação Principal

A documentação principal permanece no `README.md` da raiz do projeto, que serve como ponto de entrada principal.
