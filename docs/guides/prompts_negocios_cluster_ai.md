# 💼 Catálogo de Prompts para Negócios e Administração Empresarial - Cluster-AI

## 🎯 Guia Rápido de Utilização

### Configurações Recomendadas
- **Temperatura Baixa (0.1-0.3)**: Para análises objetivas, dados financeiros
- **Temperatura Média (0.4-0.6)**: Para planejamento estratégico, análise de mercado
- **Temperatura Alta (0.7-0.9)**: Para ideação criativa, inovação

### Modelos por Categoria
- **LLM Geral**: Llama 3, Mixtral, Mistral
- **Análise**: Mixtral, Llama 3 70B

## 📁 CATEGORIA: ESTRATÉGIA EMPRESARIAL

### 1. Análise SWOT para Cluster-AI
**Modelo**: Mixtral/Llama 3

```
[Instrução: Atue como um consultor estratégico empresarial]

Realize uma análise SWOT completa para o projeto Cluster-AI:

**Contexto atual:** [descrição da situação do projeto]
**Objetivos estratégicos:** [crescimento/expansão/adoção/etc.]
**Mercado alvo:** [empresas/pesquisadores/desenvolvedores]
**Concorrência:** [soluções similares no mercado]

**Solicito:**
1. **Forças (Strengths):** Vantagens competitivas do Cluster-AI
2. **Fraquezas (Weaknesses):** Pontos que precisam ser melhorados
3. **Oportunidades (Opportunities):** Tendências e gaps de mercado
4. **Ameaças (Threats):** Riscos externos e barreiras
5. **Plano de ação estratégico** baseado na análise
```

## 💡 DICAS PARA GESTÃO EMPRESARIAL

### Foco em Resultados
Sempre alinhe decisões com objetivos estratégicos mensuráveis

### Análise de Dados
Use dados para fundamentar decisões, não opiniões

### Agilidade Estratégica
Adapte-se rapidamente às mudanças do mercado

### Inovação Contínua
Busque constantemente oportunidades de melhoria

### Gestão de Riscos
Identifique e mitigue riscos proativamente
