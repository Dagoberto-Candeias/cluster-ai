# 🔍 Catálogo de Prompts para Auditoria e Compliance - Cluster-AI

## 🎯 Guia Rápido de Utilização

### Configurações Recomendadas
- **Temperatura Baixa (0.1-0.3)**: Para avaliações objetivas, compliance rigoroso
- **Temperatura Média (0.4-0.6)**: Para recomendações, planejamento de controles
- **Temperatura Alta (0.7-0.9)**: Para cenários hipotéticos, inovação em controles

### Modelos por Categoria
- **LLM Geral**: Mixtral, Llama 3
- **Análise Técnica**: CodeLlama, DeepSeek-Coder
